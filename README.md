# Simple JWT Login FE

## Setup

**Clone this repository locally**
```bash
https://github.com/decagondev/simpleJwtLoginFe.git
```

**Change directory**
```bash
cd simpleJwtLoginFe
```

**install dependencies**
```bash
npm install
```

**run the react application**
```bash
npm run dev
```
